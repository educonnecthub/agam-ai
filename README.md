# AGAM

> A privacy-first AI companion that remembers, understands, and grows with one person over time.

## Vision

AGAM is a lifelong AI companion designed to help one person think clearly, preserve meaningful memories, reflect deeply, and grow continuously.

## Features

- FastAPI Backend
- Modular Architecture
- Privacy-First Design
- AI Provider Abstraction (Planned)
- Long-Term Memory (Planned)
- Journal (Planned)
- Voice Interface (Planned)
